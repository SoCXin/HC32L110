#### [Level](https://github.com/SoCXin/Level)：800 MHz

## [芯片描述](https://github.com/SoCXin/V831/wiki)

[V831](https://github.com/SoCXin/V831)带有一个0.2T的NPU，cortex-A7单核，内置64M DDR2，基本跟V3S类似，主要差异是,这个芯片用QFN88来封装，跟F1C100S的一致

#### 关键特性

*  QFN88
*  NPU 0.2T
